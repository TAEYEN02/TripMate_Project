# TripMate_Project

## ✈️ 소개
사용자 맞춤 여행 일정 자동 생성 & 지도 기반 여행 계획 플랫폼

## 🛠 기술 스택
- Front: React (tripmate/)
- Back: Spring Boot (trip/)
- DB: H2 (test) / MySQL (prod)

## 📂 폴더 구조
- trip/ : 백엔드(Spring)
- tripmate/ : 프론트엔드(React)

## 🔧 브랜치 전략
- `main`: 배포용
- `dev`: 개발 통합

## 🔑 커밋 컨벤션
- [feat] 기능 추가
- [fix] 버그 수정
- [refactor] 리팩토링
